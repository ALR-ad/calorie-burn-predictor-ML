#  Calories Burn Prediction App

This is a Machine Learning web app that predicts calories burned based on user fitness data.
Data is fetched as a CSV file from Kaggle.

## Features
- Predict calories burned
- Uses Linear Regression
- Built with Streamlit

## Tech Stack
- Python
- Scikit-learn
- Streamlit

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
